# BasketGuard Analytics

Pure price-behaviour calculations for BasketGuard.

Current scope:

- year-on-year price increase;
- competitor median YoY;
- retailer excess inflation;
- current premium over cheapest equivalent;
- historical discount strength;
- shrinkflation effective increase;
- weighted offender score from normalised components.

This package is dependency-free and uses `Decimal` internally for predictable money/ratio calculations.
